# ASCEND — Project Overview

## Concept
ASCEND is a minimalist competitive roadmap platform for a private group of 8–10 people following the same AI/ML roadmap.

The platform is NOT a productivity app.
It is a clean leaderboard-based competition system.

Users:
- complete roadmap tasks
- submit LinkedIn proof
- gain XP
- climb leaderboard

The experience should feel:
- premium
- minimal
- smooth
- competitive
- modern

---

# Core Philosophy

The app should create:
- visible progress
- silent competition
- social pressure
- consistency

No unnecessary features.
No clutter.
No complicated dashboards.

---

# Main Features

## Authentication
Each user has:
- login
- profile
- XP
- progress
- streak

---

## Roadmap System

Tasks are grouped into sections:
- Foundations
- Machine Learning
- Deep Learning
- Projects

Each task contains:
- title
- description
- XP reward
- deadline
- status

Statuses:
- Locked
- Upcoming
- In Progress
- Completed

---

## Proof Submission

Users submit:
- LinkedIn post URL

Admin:
- approves or rejects

Once approved:
- XP updates
- task becomes completed
- leaderboard refreshes

---

## Leaderboard

Shows:
- rank
- XP
- progress percentage
- streak
- movement indicators

Should feel:
- elegant
- alive
- animated

---

# V1 Scope

Only build:
- login
- roadmap
- leaderboard
- proof submission
- admin approval

Avoid:
- chat
- comments
- AI moderation
- notifications
- mobile app
- analytics
