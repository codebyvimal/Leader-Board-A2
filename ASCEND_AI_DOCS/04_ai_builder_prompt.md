# ASCEND — AI Builder Master Prompt

Build a modern full-stack web application called ASCEND.

ASCEND is a minimalist competitive roadmap platform for a private AI/ML learning group.

The app should feel:
- premium
- futuristic
- smooth
- minimal
- competitive

DO NOT build a traditional productivity dashboard.

The UI style should resemble:
- Linear
- Raycast
- Apple
- minimal esports interfaces

Use:
- dark theme
- purple accent glow
- glassmorphism
- subtle motion
- clean typography
- large spacing

---

# Features

## Authentication
Users can:
- login
- access their own roadmap progress

Use Supabase authentication.

---

## Roadmap Page

Display:
- roadmap sections
- tasks
- completion states
- deadlines
- progress percentage

Task states:
- locked
- upcoming
- in progress
- completed

When clicking a task:
- open task details panel
- show description
- show deadline
- allow proof submission

---

## Proof Submission

Users paste:
- LinkedIn post URL

Store submission in database.

Admin can:
- approve
- reject

Once approved:
- mark task completed
- update XP
- refresh leaderboard

---

## Leaderboard

Display:
- rank
- user avatar
- XP
- progress percentage
- movement indicators

Leaderboard should:
- animate smoothly
- update realtime
- feel alive but minimal

---

# Technical Requirements

Use:
- Next.js App Router
- Tailwind CSS
- Framer Motion
- Supabase

Deploy-ready architecture.

---

# Important Rules

DO:
- keep everything minimal
- use smooth animations
- prioritize spacing and typography
- build responsive UI
- use reusable components

DO NOT:
- add unnecessary features
- create cluttered dashboards
- use many colors
- overcomplicate backend

The app should feel like:
“an elite private roadmap competition platform.”
