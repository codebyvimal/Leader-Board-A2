# ASCEND — UI Design System

# Design Direction

Minimal futuristic UI.

Inspired by:
- Linear
- Raycast
- Apple
- Minimal gaming interfaces

The UI should feel:
- cinematic
- smooth
- focused
- premium

---

# Color Palette (Just for suggestions, do your own if you think good)

## Background
- #050816
- #0B1020

## Accent
- Purple/Violet glow
- #7C3AED
- #8B5CF6

## Text
- White
- Soft gray

Avoid:
- rainbow gradients
- colorful dashboards
- heavy neumorphism

---

# Typography

Use:
- Inter
- Geist
- SF Pro Display

Typography should be:
- clean
- spacious
- bold headings
- light body text

---

# Animations

Animations are VERY important.

Use:
- Framer Motion
- smooth transitions
- subtle hover glow
- progress animations
- count-up XP animations

Avoid:
- exaggerated motion
- flashy gaming effects

---

# Layout

Main page layout:

LEFT:
- sidebar

CENTER:
- roadmap
- leaderboard

RIGHT:
- activity
- stats

---

# UI Rules

Keep:
- large spacing
- thin borders
- soft blur
- glass effect
- minimal text

Avoid:
- crowded cards
- large shadows
- excessive gradients
- too many colors

---

# Mood

The app should feel like:
“an elite private progress club.”
